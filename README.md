# 🚀 AVATAR STAR AI AIMBOT - HƯỚNG DẪN CHI TIẾT 🎯

Chào mừng bạn đến với bản Aimbot sử dụng trí tuệ nhân tạo (YOLOv8) dành riêng cho Avatar Star!

---

## 🛠️ Hướng dẫn cài đặt trên máy Windows (Step-by-Step)

### Bước 1: Tải Code về máy
- Nhấn vào nút xanh **Code** ở phía trên.
- Chọn **Download ZIP**.
- Giải nén thư mục vừa tải về.

### Bước 2: Cài đặt Python
- Tải Python bản 3.9 hoặc 3.10 tại: [python.org](https://www.python.org/downloads/)
- **QUAN TRỌNG:** Khi cài đặt, bạn **BẮT BUỘC** phải tích vào ô **"Add Python to PATH"** ở màn hình đầu tiên.

### Bước 3: Cài đặt các thư viện cần thiết
- Mở thư mục bạn vừa giải nén.
- Nhấn chuột phải vào vùng trống trong thư mục, chọn **"Open in Terminal"** (hoặc CMD).
- Gõ lệnh sau và nhấn Enter:
  ```bash
  pip install -r requirements.txt
  ```
- Đợi cho máy tải xong các "linh kiện" cần thiết.

### Bước 4: Chạy Tool
- Bạn chỉ cần tìm file **`run_bot.bat`** và nhấn đúp chuột vào nó.
- Nếu thấy hiện bảng đen có dòng chữ `🚀 AVATAR STAR AI AIMBOT 🚀` là bạn đã thành công!

---

## 🎮 Cách sử dụng trong Game

- **Đè phím Shift:** AI sẽ tự động "khóa" tâm súng vào đầu kẻ địch gần tâm súng nhất.
- **Nhấn Chuột Trái:** Tự động thực hiện "Quick Scope" (vẩy tâm và bắn) để đạn Sniper bay thẳng tắp.
- **Phím Q:** Nhấn để tắt Tool ngay lập tức.

---

## ⚙️ Tùy chỉnh (dành cho cao thủ)

Nếu bạn muốn thay đổi cách bot hoạt động, hãy mở file **`config.py`** bằng Notepad:
- `HEAD_OFFSET_Y = -15`: Sửa số này nếu bạn muốn tâm súng ngắm cao hơn hoặc thấp hơn đầu.
- `FOV_SIZE = 400`: Tăng số này nếu muốn bot quét vùng rộng hơn trên màn hình.
