# Avatar Star AI Aimbot - Rage Edition 🚀🎯

Bản Aimbot sử dụng trí tuệ nhân tạo (YOLOv8) được huấn luyện riêng cho game Avatar Star.

## ✨ Tính năng nổi bật
- **AI-Powered Detection:** Nhận diện nhân vật thông minh qua bộ não YOLOv8.
- **Snap Aim:** Giật tâm tức thời vào mục tiêu (Smooth = 1.0).
- **Quick Scope Macro:** Tự động vẩy Scope khi bắn Sniper để đạn bay thẳng tắp.
- **Ally Filtering:** Tự động nhận diện thanh máu xanh để không bắn nhầm đồng đội.
- **Shift-Lock:** Khóa mục tiêu khi đè phím Shift.

## 🛠️ Cách cài đặt (Dành cho Windows)
1. Cài đặt Python 3.9+
2. Mở CMD và cài đặt thư viện:
   ```bash
   pip install -r requirements.txt
   ```
3. Chạy Tool:
   - Nhấn đúp vào file `run_bot.bat` hoặc gõ:
   ```bash
   python main.py
   ```

## ⚙️ Tùy chỉnh (config.py)
- `HEAD_OFFSET_Y`: Chỉnh độ cao thấp của tâm súng (mặc định -15 cho headshot).
- `FOV_SIZE`: Kích thước vùng quét ảnh.
